# 🚀 Portfolio Website – Yesh V3.0 (React Version)

## 📖 Description

Portfolio Yesh V3.0 is the latest and most advanced version of my personal portfolio website built using **React and Vite**. It is designed to showcase my skills, projects, certifications, and experience in a modern, dynamic, and highly interactive way.

This version significantly improves performance, scalability, and UI/UX compared to previous static versions.

---

## 🎯 Problem Statement

Traditional portfolio websites built with basic technologies often lack scalability, performance optimization, and dynamic interactivity.

This project solves these problems by:

* Using React for component-based architecture
* Improving performance with Vite
* Creating a modern and scalable portfolio
* Enhancing user experience with dynamic UI

---

## 🚀 Features

* ⚛️ Built with React (Component-based architecture)
* ⚡ Fast performance using Vite
* 🎯 Modern hero section with animations

* 📂 Dynamic project showcase

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/Yeshwanth1953/Portfolio_Yesh_V3.0.git

2. Navigate to the project folder:
   cd Portfolio_Yesh_V3.0

3. Install dependencies:
   npm install

4. Run the project:
   npm run dev

---

## ▶️ Usage

* Run the development server
